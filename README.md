
# 🤖 AI Data Visualization Agent (Superstore Analytics)

An AI-powered data visualization project that automatically analyzes a dataset and generates business insights and charts.

This project is designed as a **portfolio-ready GitHub + LinkedIn viral project** for aspiring **Data Analysts / AI Analysts**.

---

## 🚀 Project Idea

Traditional dashboards require manual analysis.

This project builds an **AI Data Visualization Agent** that:

• Reads a dataset  
• Performs automatic exploratory data analysis  
• Generates charts  
• Explains insights like a business analyst  

Think of it as a **mini AI analyst for your dataset**.

---

## 📊 Dataset

Superstore Canada Dataset

Features:
- Orders
- Sales
- Profit
- Discounts
- Regions
- Product categories

Rows: ~10,000

---

## 📈 Example Insights

### 1️⃣ Sales by Category
Shows which product category drives the most revenue.

### 2️⃣ Profit by Region
Identifies most profitable regions.

### 3️⃣ Monthly Sales Trend
Shows seasonality and growth trends.

## 🧠 AI Agent Workflow
